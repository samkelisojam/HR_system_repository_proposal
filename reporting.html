Reporting Microservice

Overview

This microservice generates reports for administration based on attendance and payroll data.

APIs

Method Endpoint Description Request Body Response Body
GET /reports/attendance Generate attendance reports. N/A { "attendanceReport": [...] }
GET /reports/salary Generate salary reports. N/A { "salaryReport": [...] }

Pseudocode for Methods

        // Generate attendance report
        function generateAttendanceReport() {
            // Fetch attendance data from database
            // Format data into report
            // Return report
        }

        // Generate salary report
        function generateSalaryReport() {
            // Fetch salary data from database
            // Format data into report
            // Return report
        }

Interactions

This microservice pulls data from the Payroll Microservice and Clocking System Microservice to generate reports.
