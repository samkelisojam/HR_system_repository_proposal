Clocking System Microservice

Overview

This microservice tracks employee attendance and clocking in/out times.

APIs

Method Endpoint Description Request Body Response Body
POST /clock-in Record clock-in time. { "employeeId": 1 } { "message": "Clock-in recorded successfully." }
POST /clock-out Record clock-out time. { "employeeId": 1 } { "message": "Clock-out recorded successfully." }
GET /attendance/{employeeId} Retrieve attendance records. N/A { "employeeId": 1, "attendance": [{ "date": "2023-01-01", "status": "Present" }] }
