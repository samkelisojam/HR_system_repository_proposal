Employee Agreement Microservice

Overview

This microservice manages employee agreements and handovers.

APIs

Method Endpoint Description Request Body Response Body
POST /agreements Submit an employee agreement. { "employeeId": 1, "agreement": { "type": "Employment", "details": "Full-time employee agreement." } } { "message": "Agreement submitted successfully." }
GET /agreements/{employeeId} Retrieve agreements for an employee. N/A { "employeeId": 1, "agreements": [{ "type": "Employment", "details": "Full-time employee agreement." }] }
