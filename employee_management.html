Employee Management Microservice

Overview

This microservice is responsible for managing employee records, including creating, updating, and deleting employee information.

APIs

Method Endpoint Description Request Body Response Body
POST /employees Create a new employee. { "name": "John Doe", "email": "john.doe@example.com", "department": "HR" } { "id": 1, "message": "Employee created successfully." }
GET /employees/{id} Retrieve employee details. N/A { "id": 1, "name": "John Doe", "email": "john.doe@example.com", "department": "HR" }
PUT /employees/{id} Update employee information. { "name": "John Smith", "email": "john.smith@example.com" } { "message": "Employee updated successfully." }
DELETE /employees/{id} Remove an employee record. N/A { "message": "Employee deleted successfully." }
