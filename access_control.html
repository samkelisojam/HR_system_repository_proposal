Access Control Microservice

Overview

This microservice manages access to various systems for employees.

APIs

Method Endpoint Description Request Body Response Body
POST /access-request Request access to a system. { "employeeId": 1, "system": "HR System" } { "message": "Access request submitted successfully." }
GET /access/{employeeId} Retrieve access permissions. N/A { "employeeId": 1, "access": ["HR System", "Payroll System"] }
