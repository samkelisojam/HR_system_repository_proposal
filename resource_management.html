Resource Management Microservice

Overview

This microservice manages resources required for employees.

APIs

Method Endpoint Description Request Body Response Body
GET /resources Retrieve available resources. N/A { "resources": ["Laptop", "Desk", "Chair"] }
POST /resources/request Request resources. { "employeeId": 1, "resourceId": "Laptop" } { "message": "Resource request submitted successfully." }

Interactions

This microservice interacts with the Employee Management Microservice to ensure resources are assigned to active employees.
