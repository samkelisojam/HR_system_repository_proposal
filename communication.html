Communication Microservice

Overview

This microservice facilitates communication between employees and management.

APIs

Method Endpoint Description Request Body Response Body
POST /messages Send a message. { "senderId": 1, "receiverId": 2, "content": "Hello, please review my leave request." } { "message": "Message sent successfully." }
GET /messages/{employeeId} Retrieve messages for an employee. N/A { "employeeId": 1, "messages": [{ "from": 2, "content": "Hello, please review my leave request." }] }
